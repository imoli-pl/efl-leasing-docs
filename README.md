# EFL Documentation

[![Nuxt UI](https://img.shields.io/badge/Made%20with-Nuxt%20UI-00DC82?logo=nuxt&labelColor=020420)](https://ui.nuxt.com)

Documentation for the EFL project, built with [Nuxt UI](https://ui.nuxt.com) and [Nuxt Content](https://content.nuxt.com).

## Setup

Install dependencies:

```bash
pnpm install
```

## Development Server

Start the development server at `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```

See the [Nuxt deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
